# apk_puller

a script to pull apk from android device

## usage for apk_puller.sh

**to pull a single apk:**

./apk_puller.sh pull <package_name>

**to pull all apks:**

./apk_puller.sh all

## usage for apk_puller_win.ps1

**to pull a single apk:**

.\apk_puller_win.ps1 <package_name>

**to pull all apks:**

.\apk_puller_win.ps1 all
